Fact!
Spitfire has been under development for over a year now. It's feature set includes a full MVC pattern, a database ORM that supports relations and lots of other features.
About Spitfire Spitfire is a completely free and open source PHP framework desgined to make coding faster, more efficient and fun. The project is guided by Magic3W, a small company focused on web development.
Beware of the beta At this time Spitfire is under heavy development and it's use in big production environments is discouraged.
Spitfire for beginners This software is not ready for beginners, we try to be as helpful as possible and to fix the bugs anybody who uses SF comes across as quickly as we can. But you should know how to explain the bug correctly to us.
